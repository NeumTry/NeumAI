---
title: 'HTMLLoader'
description: 'HTMLLoader is designed to efficiently load and parse HTML files into NeumDocument objects for further processing within the Neum Ai ecosystem.'
---

The `HTMLLoader` class is tailored for ingesting HTML files, converting the data within into structured `NeumDocument` objects. This class utilizes the `UnstructuredHTMLLoader` to facilitate the loading and parsing process.

## Properties

Required properties:
- None

Optional properties:
- None
